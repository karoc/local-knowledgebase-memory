## 5. 上线配置建议（默认值）

在 memory 插件配置中建议：

```json
{
  "minRecallScore": 0.3,
  "sessionTtlDays": 7,
  "rankingMode": "hybrid",
  "rankingDebug": false
}
```

说明：
- 初次上线建议 `hybrid`
- 若观察到召回行为波动，可临时回退：
  - `rankingMode: "legacy"`
